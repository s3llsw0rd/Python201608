More Modifying Arrays

In the console:

  • minToFront(arr)
  • reverse(arr)
  • rotate1(arr, shift) AND rotate2(arr, shift)
    • 'shift' tells the function how many spaces to shift each value to the right.
    • 'shift' must be positive
    • values wrap around
  • rotate3(arr, shift)
    • 'shift' tells the function how many spaces to shift each value.
    • 'shift' can be either negative or positive
    • values wrap around

The variable arr contains an array of random length filled with random numbers. Run randomarr(len) where len is the length of the array that you want in order to get a new one.
